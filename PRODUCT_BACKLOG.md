## Definition of Done

Задача вважається виконаною, коли:

- [ ] Код написано та відрефакторено
- [ ] Unit тести написані (покриття > 75%)
- [ ] E2E тести написані для критичних шляхів
- [ ] Код відрецензовано
- [ ] Лінтування проходить без помилок
- [ ] Документація оновлена
- [ ] Функціональність протестована вручну
- [ ] Код замерджено в основну гілку

---

## Метрики успіху

- **Швидкість розробки:** X story points на sprint
- **Якість коду:** Покриття тестами > 75%
- **Продуктивність:** Время загрузки сторінки < 2 секунд
- **Надійність:** Uptime > 99.9%
- **Задоволеність користувачів:** NPS > 50
